Discovery & Inventory

Connect Stripe, Plaid, AWS API Gateway, DynamoDB, or any REST endpoint to inventory API keys, service accounts, and OAuth clients. Our connectors normalize metadata and tag owners automatically.

Risk Scoring Engine

Apply heuristics + light ML derived from 42Crunch vulnerabilities: abnormal geography, unusual request patterns, AI-driven credential stuffing. Scores refresh every 10 minutes with human-readable explanations.

Automated Playbooks

Rotate keys with one click, throttle API usage, or route suspicious tokens into a sandbox. All actions are logged in a compliance PDF with citations to BDO/Visa references so auditors are satisfied.

Technical Architecture

FastAPI backend + React dashboard. Data stored in Postgres, with connectors calling Stripe/ Plaid/ AWS APIs. Risk scoring combines open-source heuristics and threat intel input (42Crunch + Visa token predictions). Rotation runs via AWS SDK and gateway APIs. Deploy via Docker Compose or Fargate for rapid prototypes, keeping build costs under $100/mo per pilot.

  • 1️⃣ Discovery agent tails CloudWatch/Stripe logs without installing agents.
  • 2️⃣ Score anomalies (geo drift, reuse, AI-driven requests) against 42Crunch dataset.
  • 3️⃣ Rotate keys automatically, log actions for compliance, integrate alerts via Slack + email.
