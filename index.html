Problem

42Crunch’s 2026 API Security report shows schemes like API schema drift and broken input validation as top attacker paths, yet small teams lack tooling.

Solution

CipherCoverage crawls published APIs, compares the current schema to the previous snapshot, scores vulnerability, and spikes alerts with remediation playbooks.

Product & Tech

GitHub webhook + CI runner, serverless crawler, AI heuristics from the 42Crunch dataset, and a UI summarizing drift + attack risk.

Rust-based crawler, Python scoring, hosted on Cloudflare Workers + Supabase; prototypes in 48 hours using GitHub Actions and Vercel for dashboards.
